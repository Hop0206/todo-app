# 📝 To-Do List App

## 📖 Description
This is a simple Python command-line application that helps users manage daily tasks.  
You can add tasks, view your task list, and delete tasks easily.

This project is designed for beginners to practice Python basics such as:
- Variables
- Lists
- Functions
- Loops
- User input handling

---

## 🚀 Features
- ➕ Add new tasks
- 📋 View all tasks
- 🗑️ Delete tasks
- ⚠️ Error handling for invalid input
- 🧠 Simple and easy-to-use interface

---

## 🛠️ How It Works
- Tasks are stored in a list while the program is running
- The user interacts with a menu system
- Each option performs a specific function:
  - Add → saves a task
  - View → displays all tasks
  - Delete → removes a selected task
